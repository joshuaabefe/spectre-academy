# Colour Palette

## Palette Link

[Spectre Academy Palette](https://coolors.co/050913-10182e-182849-6ef2ff-eafcff)

## Core Colours

- `#050913` Deep Midnight
- `#10182E` Spectre Navy
- `#182849` Shield Blue
- `#6EF2FF` Electric Cyan
- `#EAFCFF` Frost White

## Usage

- Deep Midnight and Spectre Navy are used for the brand background and header.
- Shield Blue supports layered contrast and borders.
- Electric Cyan highlights calls to action, glow accents, and brand energy.
- Frost White supports icon details and contrast on dark surfaces.
